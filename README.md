# 🏦 Bank Compliance Tracker

A comprehensive web-based compliance management system designed for banks and financial institutions to streamline regulatory compliance, task management, employee monitoring, audit tracking, document management, and reporting.

---

## 📌 Project Overview

Bank Compliance Tracker is an enterprise-level compliance management platform developed using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**. The system helps organizations assign compliance tasks, track employee performance, monitor regulatory deadlines, maintain audit logs, manage documents securely, and generate compliance reports.

The platform supports multiple user roles including Administrators and Employees with role-based access control and secure authentication.
